## 概述
本模板用于定义设备功耗测试方法、基线指标与优化路径，适用于[产品型号]全生命周期评估。
应说明测试目标（续航、峰值电流、待机功耗、热设计约束）与适用固件版本。
建议统一测试环境：电源设备[型号]、温度[℃]、电压[V]、采样率[Hz]、网络条件。
流程总览：场景定义 → 仪器校准 → 数据采集 → 统计分析 → 优化验证。

最佳实践：建立“功耗预算表”，将模块级功耗分解到可执行优化项。
