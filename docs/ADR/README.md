# Architecture Decision Records (ADR)

## 什么是 ADR？

ADR 记录项目中的重要技术决策，包括背景、选择和理由。帮助新成员理解"为什么这样做"。

## 如何添加新 ADR

1. 复制模板创建新文件：`ADR-{NNN}-{简短标题}.md`
2. 编号递增（当前最大：003）
3. 填写所有必须章节
4. 更新本索引
5. 提交 PR
