# UI-UX-SPEC

## 设计目标
- 采用 **Scandinavian Minimalism**（北欧极简）视觉语言。
- 强调留白与秩序感，突出家具本身材质与细节。
- 使用高品质产品图片作为主要内容承载。
- 保持导航结构简洁、可预期，减少用户认知负担。

## 信息架构
页面层级：
- 首页 → 商品列表 → 商品详情 → 购物车 → 结算 → 订单确认

全局布局：
- Header：Logo + 一级导航（家具分类）+ 购物车图标
- Footer：关于我们 / 政策条款 / 联系方式

支持路径：
- 商品搜索与筛选（材质、尺寸、价格区间）
- 用户中心（订单历史、地址簿）

## 交互流程
核心购物流程：
1. 用户浏览首页与推荐商品。
2. 进入商品详情，选择颜色/尺寸等变体。
3. 点击“加入购物车”，系统更新 Mini Cart 数量与金额。
4. 进入购物车页，确认数量与配送国家。
5. 在结算页填写收货地址与联系方式。
6. 选择配送方式（标准/加急）。
7. 选择支付方式并完成支付。
8. 跳转订单确认页，展示订单号、商品明细、后续通知说明。

## 视觉规范
- 配色：中性色为主（黑、白、灰），辅以木质暖色（Oak/Birch tone）。
- 字体：系统字体栈，优先 `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`。
- 间距系统：8px 网格（8/16/24/32/48）。
- 响应式断点：
  - Mobile：`< 768px`
  - Tablet：`768px - 1023px`
  - Desktop：`>= 1024px`
- 组件风格：圆角克制、阴影轻量、按钮状态反馈清晰（hover/active/disabled）。
