# SLA

## 服务范围

本 SLA 适用于 NordHjem 电商平台所有面向用户的核心服务：

- Store 前端（Next.js）
- Store API（Medusa.js v2）
- Admin Panel（运营后台）

## 可用性目标

- 月度总体可用性目标：**99.5%**。
- 排除计划维护窗口：每月最多 2 小时（需提前公告）。

## 响应与修复时限

| 级别 | 响应时限 | 修复时限 |
| --- | --- | --- |
| P0 | 5 分钟 | 4 小时 |
| P1 | 30 分钟 | 8 小时 |
| P2 | 4 小时 | 48 小时 |
| P3 | 24 小时 | 下个 Sprint |

## 例外条款

以下场景不纳入 SLA 违约范围：

- 第三方服务故障（Stripe / Railway 等）。
- 不可抗力（自然灾害、区域性网络中断等）。
- 已提前公告的计划维护窗口。
