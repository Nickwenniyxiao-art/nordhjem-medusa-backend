# 项目范围说明

## 背景与目标

NordHjem 是面向欧洲核心受众（当前优先 DK 与 US 站点运营策略）的北欧极简风格家具电商平台，目标是在低成本条件下快速构建可上线、可转化、可运营的 DTC 业务闭环。项目目标是先交付 MVP，验证“高质量选品 + 优雅购物体验 + 稳定履约”模型，再逐步扩展品类和运营能力。

## 范围内事项

- 商品展示：分类、详情页、图片与规格展示。
- 购物车：加购、改数量、删除、价格实时计算。
- 下单结算：地址、配送方式、税费、订单确认。
- Stripe 支付：测试与生产支付链路、支付状态回写。
- 多币种：EUR / USD 展示与结算支持。
- 多地区：DK / US 区域策略、运费与税务规则差异化。
- Admin 后台：商品、库存、订单、退款基础运营能力。
- E2E 监控：关键交易路径监控（浏览→加购→支付→确认）。

## 范围外事项

- 线下门店管理与 POS 系统。
- 自建物流系统（使用第三方履约或手工履约流程）。
- B2B 批发分销体系。
- 社交电商闭环（站内直播/短视频带货）。
- 直播卖货与达人联盟体系。

## 关键假设与约束

### 关键假设

- 初期流量低于 1000 DAU，峰值并发可通过基础容器规格承载。
- 初期 SKU 与订单规模可由轻量运营流程支持。
- DK/US 两地区可通过 Region + Currency 机制满足首阶段需求。

### 关键约束

- 团队结构为单人 + AI 协作，开发与运营需高度自动化。
- 预算有限，基础设施优先采用 Railway 免费/低价方案。
- 交付优先级以“交易闭环可用”高于“高级体验增强”。
- 必须满足 GDPR 基线要求，避免过度采集用户数据。
